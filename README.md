# Smart-task-manager

A React + Vite task management dashboard built to organize tasks, schedules, habits, and productivity.

## Features

- Dashboard overview with task summaries
- Scheduler for planning tasks and time blocks
- Pomodoro timer for focused work sessions
- Habit tracker to monitor routines
- Analytics charts using Recharts
- Authentication with Login/Register and protected routes

## Getting Started

1. 
pm install
2. 
pm run dev
3. Open http://localhost:5173

## Tech Stack

- React 19
- Vite
- React Router DOM
- Recharts
- ESLint
